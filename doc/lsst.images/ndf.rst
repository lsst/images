Starlink NDF I/O
================

This is an *experimental* HDF5 serialization backend to demonstrate that the package can be used to write files in formats other than FITS.
It does write files in HDF5 format that can be read by the Starlink software.
We do not recommend you use it in a production environment at this time to create archival products.
